## Features

- CRUD
- Local data storage (`C:\Users\<username>\AppData\Roaming\js-snippets\jssDB.json`)
- Autosave new snippet and edit (Debounce)
- Create/Delete tags
- Search snippet by title or tag
- Sort date asc/desc
- Copy to clipboard
- Change settings (font size, indendation space,...)
